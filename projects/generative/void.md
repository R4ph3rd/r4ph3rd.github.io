# VOID
*Tangible creative codding*

Date: Janvier 2018

Interactive installation representing a microorganism programmed in Java.   

01. Use the QR code to go to the website                                                                                     
02. Connect to the "VOID" network                                                                                     
03. For each connection, an agent appears in the microorganism                                                                                     
04. Search the internet or watch videos to generate data                                                                                     
05. The data generated will bring the microorganism to life                                                                                     
06. As the agents' lives diminish over time, spectators must feed the organism for it to survive
